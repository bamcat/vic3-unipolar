# Overview
Welcome to UNIPOLAR, formerly 1992: A Modern Day Mod. Set in 1992, UNIPOLAR is a total conversion mod that aims to bring Victoria 3 and it's systems into the current era of politics.

# Features

- Modernized technology tree
- Modernized military units & mobilization options
- Modernized UI
- Nuclear weapons system
- International organizations system
- New diplomatic actions
- New ideology system
- Modernized map graphics
- Cleaned out vanilla content which wouldn't make sense in a 21st century context
- Starting world leaders, U.S presidents to Donald Trump (2016)
- Reworked ideologies, interest groups
- New laws
